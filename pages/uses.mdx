## Terminal

I've been trying out [Hyper](https://hyper.is/) recently - there are still some bugs here and there, but mostly it's been solid. I'll use macOS's built in Terminal.app sometimes when the situation calls for it, but Hyper is my go to these days.

I use the [Smyck color scheme](http://color.smyck.org/), and the following Hyper plugins:

1. [hyperterm-blink](https://github.com/amio/hyper-blink)
2. [hypercwd](https://github.com/hharnisc/hypercwd)
3. [hyperlinks](https://github.com/zeit/hyperlinks)
4. [hyperterm-bold-tab](https://github.com/dawsonbotsford/hyperterm-bold-tab)
5. [hyperterm-tab-icons](https://github.com/dfrankland/hyperterm-tab-icons)
