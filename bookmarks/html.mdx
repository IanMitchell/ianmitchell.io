title: HTML5
---

1. **[HEAD tag elements](https://github.com/joshbuchea/HEAD)** - An exhaustive list of what goes in the `<head>` tag.
2. **[Offline Webapps](https://github.com/pazguille/offline-first/blob/master/README.md)** - A list of resources on how to make your webapp work offline.
3. **[HTML Schemas](http://schema.org/docs/schemas.html)** - A guide on how to use schemas for different sets of structured data.
4. **[Form Autofill](https://medium.com/@grigs/autofill-what-web-devs-should-know-but-dont-d8f32575253c#.vq8d5tpo7)** - Tips on how to guide the browser in autofilling form fields. This is particularly useful since form autofilling doesn't have a lot of documentation.
5. **[Form Autcomplete](https://css-tricks.com/snippets/html/autocomplete-off/)** - Additional resources on specifying autocorrect and autocapitalization attributes.
6. **[Mobile Keyboards](http://baymard.com/blog/mobile-touch-keyboards)** - A set of tips and tricks to help indicate to mobile browsers what keyboards to use for different input types.
7. **[Form Validation](http://webdesign.tutsplus.com/tutorials/html5-form-validation-with-the-pattern-attribute--cms-25145)** - How to use the pattern attribute and style error messages with CSS and JavaScript.
8. **[Form Validation](http://alistapart.com/article/forward-thinking-form-validation)** - An article that looks at HTML5 form validation options.
9. **[Picture Element](http://www.html5rocks.com/en/tutorials/responsive/picture-element/)** - A introduction to the `<picture>` element and what its intended usage is.
10. **[When to not use Picture](http://blog.cloudfour.com/dont-use-picture-most-of-the-time/)** - When and where to not use the `<picture>` element.
11. **[Picture Snippets](https://dev.opera.com/articles/responsive-images/)** - Additional technical information on using the `<picture>` element.
12. **[Viewport](https://css-tricks.com/probably-use-initial-scale1/)** - Snippet for correctly setting the viewport on responsive designs.
13. **[Color Input via Datalist](http://codepen.io/estelle/pen/rigBl)** - Snippet for choosing a color from a set defined in a datalist.
14. **[Prevent Chrome Translation](https://davidwalsh.name/prevent-chrome-from-translating-page)** - David Walsh shows how to prevent Chrome from prompting the user to translate the page.
15. **[Plyr](https://plyr.io/#video)** - A well designed HTML5 media player.
